Sports Booking Slot is amazing free plugin that enables one to book and purchase available slots for any sports( indoor & outdoor ) and virtual games from your wordpress website. And this is 100% compatible with woocommerce.

== Description ==

Sports Booking Slot plugin empower your site to create slots for each and every sports/game, for as many as specific days you want. The visitors
can see lists of steps and options to proceed such as facilities like turf one or turf two, then available hourly slots for conjugated 3 days as per the facility selected.

so for visitor the steps are,
 * Select a facility like [sports_name] turf one or two
 * Select hourly slots for required dates
 * Proceed to payment
 * cart->checkout->woocommerce payment->order completed->slots are booked permanently.
 * Visitor can select any specific date before selecting any facility and then go for the steps above

For admin
 * Create a product from woocommerce product named like [sports_name] turf one or two
 * Go to 'Sports Booking > add new slot'
 * Select a product, date and available slots per each hour and submit it
 * Go to 'Sports Booking > Booking Slot Lists'
 * Lists of all slots you have created yet
 * Select to See, Edit and Update any created single slot here with beautiful ajax interface without refreshing the page.
 * See booked slots by visitors in 'woocommerce > orders'
 * On clicking any particular order you can see which slots are booked by the visitor.

> Must have woocommerce installed and activated previously.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sports-booking-slot` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Make sure to install and activate woocommerce plugin.

== Frequently Asked Questions ==

= can this plugin be used for multisite ? =

No.

= Can it work without woocommerce ? =

No.

= Is this functional for multiple venus ? =

No, this is not yet configured for multiple venus. Only single venue. Later update will include the use of multiple venus. but
technically, you can assign multiple venue assigning product name like this [Venue_name]_[Sports_name] turf one or two or as many as you need.
